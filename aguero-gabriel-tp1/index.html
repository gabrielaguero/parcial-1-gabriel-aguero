Bienvenidos melómanos a música manía!

Nuestra API proporciona acceso a una vasta colección de información sobre artistasy álbumes. Ya sea que estés construyendo una aplicación para descubrir nueva música, crear listas de reproducción personalizadas o explorar datos relacionados con tus artistas favoritos, nuestra API tiene todo lo que necesitas.

Endpoints

  • Get

    Endpoint: /discos

    Obtienes todos los discos

  • Post

    Endpoint: /discos

    Crear un nuevo disco

  • Get

    Endpoint: /discos/:id

    Obtienes todos los discos por su ID

  • Put

    Endpoint: /discos/:id

    Actualizar los datos del disco

  • Delete

    Endpoint: /discos/:id

    Elimminas un disco

  • Get

    Endpoint: /discos/:nombre

    Buscas los discos por su nombre

También pudes obtener la información de las bandas

  • Get

    Endpoint: /bandas

    Obtienes todos los bandas

  • Get

    Endpoint: /bandas/:año

    Obtienes las bandas ordenadas por año

  • Put

    Endpoint: /bandas/:id

    Actualizar los datos de la banda

  • Delete

    Endpoint: /bandas/:id

    Elimminas una banda

  • Post

    Endpoint: /bandas

    Crear una nueva banda
